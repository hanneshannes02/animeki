# AnimeKI

Statische, SEO-optimierte Anime-Website auf Deutsch.

## Direkt online stellen

1. Alle Dateien auf dein Webhosting in das Webroot laden.
2. Domain auf das Webroot zeigen lassen.
3. Falls Domain nicht `animeki.de` ist, in allen HTML-Dateien `canonical` und in `robots.txt` + `sitemap.xml` die URL anpassen.

## Inhalt

- `index.html` - Startseite mit Suchintentionen + FAQ + Schema.org
- `anime-streaming-deutschland.html` - Landingpage für Streaming-Keywords
- `anime-webseiten-uebersicht.html` - Landingpage für wichtigste Anime-Webseiten
- `anime-news-und-trends.html` - Landingpage für News-Keywords
- `anime-saisonkalender-2026.html` - Landingpage für Saison-Keywords
- `anime-einstieg-guide.html` - Landingpage für Einsteiger-Keywords
- `styles.css` - Layout/Design
- `script.js` - kleine Suchhervorhebung über `?q=...`
- `robots.txt` - Crawling-Hinweis
- `sitemap.xml` - Sitemap für Suchmaschinen
